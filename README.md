# Lure.mdsvex

Trying MDSveX for publishing LURE material.

## Specs (meta)

- no native tools to be installed (run via Multipass)
- OS either macOS or Windows + WSL2

## Requirements

- [Multipass](https://multipass.run) installed

> Developed on:
> 
> macOS 13.4


## Usage

```
$ make
```

Builds the output.
